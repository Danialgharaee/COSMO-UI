# 🧑🏻‍🚀COSMO UI
Streamlit UI for 🧑🏻‍🚀COSMO

- Get your Hugging Face API key
- Replace that key in the `config.py` code
- Install the required libraries `pip install -r requirements.txt`
- Run `streamlit run app.py`
